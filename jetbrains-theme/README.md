# NERV Theme — JetBrains IDEs

A dark theme for IntelliJ IDEA, PyCharm, WebStorm, and all JetBrains IDEs.

## Installation

### From JAR (manual)

1. Download `nerv-theme.jar` from [Releases](https://github.com/LynnYuSmith/nerv-theme/releases)
2. In your IDE: **Settings → Plugins → ⚙️ → Install Plugin from Disk...**
3. Select the `.jar` file and restart
4. Go to **Settings → Appearance & Behavior → Appearance** → select **NERV**

### Build from source

```bash
cd jetbrains-theme
./build.sh
```

This produces `nerv-theme.jar` in the current directory.

## Compatibility

- IntelliJ IDEA 2023.3+
- PyCharm 2023.3+
- WebStorm 2023.3+
- All other JetBrains IDEs based on IntelliJ Platform 2023.3+
